  • length：ArrayList申请的内容空间长度

解析：最后grow()方法中

  • oldCapacity >> 1表示oldCapacity 右移一位，其效果相当于oldCapacity /2，即ArrayList默认扩容至1.5倍
  • 调用Arrays.copyOf方法将elementData数组指向新的容量为newCapacity的连续空间并将elementData的数据复制到新的内存空间
